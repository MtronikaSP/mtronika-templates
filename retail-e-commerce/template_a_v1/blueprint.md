# Retail E-Commerce template_a_v1

## Template Style: Ashion - Fashion E-Commerce
- **Framework**: Bootstrap 4
- **Typography**: Cookie + Montserrat (Google Fonts)
- **Theme**: Clean white + Black accents + Minimal design

---

## Sections

- **Preloader**: Animated loading spinner
- **Offcanvas Menu**: Mobile slide-out menu with icons, logo, nav links, Login/Register
- **Header**: Logo + Main navigation (Home, Women's, Men's, Shop, Pages dropdown, Blog, Contact) + Login/Register + Search/Wishlist/Cart icons with badge counts
- **Categories/Hero**: 6-grid hero layout (Large Women's fashion banner + 4 smaller category boxes: Men's, Kid's, Cosmetics, Accessories) with "Shop now" CTAs
- **Product Grid**: Title "New product" + Category filter tabs (All, Women's, Men's, Kid's, Accessories, Cosmetics) + 8 product cards with hover actions (expand, heart, cart), ratings, prices, sale/new/stockout badges
- **Banner Slider**: Full-width background image + Owl Carousel text slides ("The Chloe Collection")
- **Trend Sections**: 3-column layout (Hot Trend, Best Seller, Feature) with small product cards
- **Discount Section**: Split layout with product image + Countdown timer + "Shop now" CTA
- **Services/Guarantees**: 4-column icons (Free Shipping, Money Back, 24/7 Support, Payment Secure)
- **Instagram Gallery**: 6-column Instagram image grid with hover overlay
- **Footer**: Logo + Description + Payment icons + Quick links + Account links + Newsletter subscription + Social icons
- **Search Modal**: Fullscreen search overlay
