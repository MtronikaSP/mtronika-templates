# Education template_a_v1

## Template Style: EduWell - Education/School
- **Framework**: Bootstrap 4
- **Typography**: Open Sans (Google Fonts)
- **Theme**: Purple/Pink gradient accent with modern flat design

---

## Sections

- **Header/Navbar**: Sticky header + Logo image + Navigation (Home, Services, Courses, Pages dropdown, Testimonials, Contact)
- **Hero/Banner**: ID `#top` (Left: Welcome text + Headline + Gradient CTA button | Right: Illustration image)
- **Services**: ID `#services` (Owl Carousel slider with icon cards: Useful Tricks, Creative Ideas, Ready Target, Technology)
- **Courses**: ID `#courses` (Left sidebar tab menu + Right content area with course details, price, duration, certificates, Subscribe CTA)
- **Simple CTA**: Sale promotion section (50% OFF banner with illustration + white CTA button)
- **Testimonials**: ID `#testimonials` (Owl Carousel quotes with names + titles + quote icon)
- **Contact Section**: ID `#contact-section` (Google Maps embed + Phone/Mobile info cards + Contact form: Name, Email, Message + Gradient submit button)
- **Footer**: Social icons row + Copyright
