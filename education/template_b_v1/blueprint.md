# Education template_b_v1

## Template Style: Kider - Kindergarten/Preschool
- **Framework**: Bootstrap 5
- **Typography**: Heebo + Inter + Lobster Two (Google Fonts)
- **Theme**: Colorful primary palette (Blue, Green, Yellow, Orange) with playful rounded elements

---

## Sections

- **Spinner/Loader**: Animated loading spinner
- **Navbar**: Sticky nav + Icon logo ("Kider") + Navigation + "Join Us" pill button
- **Hero Carousel**: Owl Carousel with fullscreen images + Headlines + Dual CTAs (Learn More + Our Classes)
- **Facilities**: 4-column colored icon cards (School Bus, Playground, Healthy Canteen, Positive Learning)
- **About Section**: Left content (Headline + Description + Read More CTA + Founder info) + Right circular images layout
- **Call To Action**: "Become A Teacher" banner with image + Get Started CTA
- **Classes/Courses**: 6 class cards with circular images, teacher info, price pill, Age/Time/Capacity details
- **Appointment Form**: Left form (Guardian Name, Email, Child Name, Age, Message) + Right image
- **Team/Teachers**: 3 teacher cards with circular images + social links
- **Testimonials**: Owl Carousel rounded cards with client quotes
- **Footer**: Dark footer with 4 columns (Get In Touch, Quick Links, Photo Gallery grid, Newsletter signup)
