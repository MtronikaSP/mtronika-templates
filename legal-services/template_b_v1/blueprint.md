# Legal Services Template B - Finanza (Financial Services)

## Template Overview
- **Theme**: Financial/Legal Consulting Services
- **Template Name**: Finanza
- **Primary Color**: Blue (#primary)
- **Fonts**: Jost, Open Sans (Google Fonts)
- **Framework**: Bootstrap 5

## Page Sections

### 1. Navbar (Fixed)
- **Structure**: Fixed top with topbar
- **Topbar**: Contact info (address, hours, email, phone)
- **Logo**: Text-based "Finanza"
- **Navigation**:
  - Home, About, Services
  - Pages Dropdown (Projects, Features, Team, Testimonial, 404)
  - Contact
- **Social Icons**: Facebook, Twitter, LinkedIn

### 2. Hero Carousel
- **ID**: `#header-carousel`
- **Type**: Bootstrap carousel with fade effect
- **Slides**: 2 slides with background images
- **Content**: Badge, headline, CTA button
- **Classes**: `carousel-item`, `carousel-caption`

### 3. About Section
- **Layout**: 2-column (image + content)
- **Features**:
  - Image on left
  - Tabbed content (Story, Mission, Vision)
  - Feature cards below (No Hidden Cost, Dedicated Team, 24/7 Available)
- **Animation**: `wow fadeInUp`

### 4. Facts Section
- **Background**: Full-width with overlay
- **Layout**: 4-column statistics
- **Content**: Counter-up animation
- **Stats**: Happy Clients, Projects Completed, Dedicated Staff, Awards

### 5. Features Section
- **Class**: `feature`
- **Layout**: 2-column grid
- **Content**:
  - Left: Intro text with CTA
  - Right: Feature boxes (Fast Executions, Guide & Support, Financial Secure)
- **Cards**: Bordered with icons

### 6. Services Section (Tabbed)
- **Class**: `service`
- **Layout**: Sidebar tabs + content area
- **Tabs**:
  - Financial Planning
  - Cash Investment
  - Financial Consultancy
  - Business Loans
- **Tab Content**: Image + description + bullet points

### 7. Callback/Contact Form
- **Class**: `callback`
- **Background**: Overlay with image
- **Form Fields**: Name, Email, Mobile, Subject, Message
- **Layout**: Centered card with form

### 8. Projects Carousel
- **Class**: `project-carousel`
- **Type**: Owl Carousel
- **Items**: Project cards with image and title

### 9. Team Section
- **Layout**: 3-column grid
- **Cards**: Image with hover overlay
- **Social**: Facebook, Twitter, Instagram

### 10. Testimonials Carousel
- **Class**: `testimonial-carousel`
- **Type**: Owl Carousel
- **Layout**: Quote + client image + name/profession

### 11. Footer
- **Background**: Dark
- **Columns**:
  - Our Office (address, contact, social)
  - Services links
  - Quick Links
  - Newsletter signup
- **Copyright**: Bottom bar

## Dependencies
- Bootstrap 5.0
- jQuery 3.4.1
- WOW.js (animations)
- Owl Carousel
- Counter Up
- Font Awesome 5
- Bootstrap Icons

## Styling Notes
- Uses `wow fadeIn` animations throughout
- Primary color theme (customizable via CSS)
- Responsive design with Bootstrap grid
- Modern tabbed interfaces for services and about
