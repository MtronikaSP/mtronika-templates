# Healthcare-Medical template_a_v1

## Template Style: Klinik - Medical Clinic
- **Framework**: Bootstrap 5
- **Typography**: Open Sans + Roboto (Google Fonts)
- **Theme**: Medical blue primary color with clean professional look

---

## Sections

- **Topbar**: Location + Hours + Phone + Social Icons
- **Navbar**: Logo + Navigation (Home, About, Service, Pages dropdown, Contact) + Appointment CTA
- **Header/Hero**: Stats counter (Doctors, Staff, Patients) + Image Carousel (Cardiology, Neurology, Pulmonary)
- **About Section**: Dual image layout + Trust content + Bullet checklist + Read More CTA
- **Services**: 6-card grid (Cardiology, Pulmonary, Neurology, Orthopedics, Dental Surgery, Laboratory)
- **Features/Why Choose Us**: Primary background banner with 4 feature icons (Doctors, Services, Consultation, Support)
- **Team/Doctors**: 4-column team grid with social links
- **Appointment**: Left info block + Right appointment booking form
- **Testimonials**: Owl Carousel patient reviews
- **Footer**: 4-column layout (Address, Services, Quick Links, Newsletter)
