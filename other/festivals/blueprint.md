# Festivals Template

## Template Style: Event / Festival
- **Framework**: Bootstrap 5.2.2
- **Typography**: Outfit (Google Fonts)
- **Theme**: Primary Yellow (#F8CB2E), Secondary Orange (#EE5007), Dark Backgrounds

---

## Sections
- **Navbar**: Sticky transparent-to-dark navigation
- **Hero Section**: Centered "Night Live" branding, Date/Location info, Background Video
- **About**: "About Festava" text block + Image with overlay info
- **Artists**: Grid of artist cards with hover effect revealing details (Name, Birthdate, Music, Youtube)
- **Schedule**: Table layout with color-coded or image-background cells for different events/times
- **Pricing**: Ticket plans (Early Bird, Standard) with savings tag and feature list
- **Contact**: Tabbed section switching between Contact Form and Google Map
- **Footer**: Top social bar, multi-column links, contact info, copyright

---

## Additional Pages
- `index.html` (Main Page)
- `ticket.html` (Dedicated ticket booking form page)

---

## Key Features
- **Video Background**: HTML5 Video in Hero
- **Tabbed Content**: For Contact section (Form vs Map)
- **Smooth Scroll**: Navigation links
- **Click Scroll**: Active section highlighting
