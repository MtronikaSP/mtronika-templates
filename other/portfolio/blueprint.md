# Portfolio Template

## Template Style: Personal Portfolio / Landing Page
- **Framework**: Bootstrap 4.6.0
- **Typography**: Open Sans, Baloo Paaji (Google Fonts)
- **Theme**: Multi-color (Blue, Indigo, Purple, Pink, Red, Orange, Yellow, Green, Teal, Cyan) defined in root variables.
- **Style**: Single Page Application (SPA) feel with scroll spy.

---

## Sections

- **Navbar**: Custom sticky navbar with hamburger menu
- **Header**: Hero section with overlay, title, subtitle, and CTA ("Visit My Works")
- **About**: "Who Am I?" section with image and text
- **Service**: "What I Do?" section with 4 service cards (Icons + Title + Desc)
- **Portfolio**: "What I Did?" section with portfolio cards (Image + Overlay Caption)
- **Pricing**: "How Much I Charge?" section with 3 pricing tiers (Free, Basic, Premium)
- **Hire Me**: CTA banner "Want to work with me?"
- **Testimonial**: Client feedback cards
- **Blog**: Recent posts section
- **Contact**: Contact form "How can you communicate?"
- **Footer**: Copyright and Social Links

## Key Features
- **Scroll Spy**: Navigation updates based on scroll position
- **Affix**: Sticky navigation behavior
- **Themify Icons**: Icon set used throughout
- **Hamburger Menu**: Mobile navigation toggle
