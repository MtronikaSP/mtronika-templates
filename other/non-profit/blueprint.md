# Non-Profit Template

## Template Style: Charity / Non-Profit
- **Framework**: Bootstrap 5.2.2
- **Typography**: Metropolis (Custom Font)
- **Theme**: Green (#5bc1ac), Secondary Blue-Grey (#5a6f80), Dark Footer (#44525d)

---

## Sections

- **Navbar**: Sticky navigation with "Kind Heart Charity" branding
- **Hero Slider**: Full-screen carousel with captions and "Discover More" / "Donate" buttons
- **Feature**: 3-column feature blocks (e.g., Become a Volunteer, Caring Earth) with hover effects
- **About**: "Our Story" with image, text, and counter statistics
- **Make an Impact**: CTA section with buttons
- **Volunteer**: "Become a Volunteer" form section
- **Causes**: "Featured Cause" or "Recent Causes" with donation progress bars (implied from CSS)
- **Donate**: Dedicated donation form with frequency selection (One-time/Monthly)
- **News**: Latest news/blog posts grid
- **Testimonial**: "Happy Customers" carousel
- **Contact**: Contact form and information details
- **Footer**: Quick links, Contact info, Social icons, Copyright

## Key Features
- **Sticky Navbar**: jQuery Sticky plugin
- **Click Scroll**: Smooth scrolling navigation
- **Counter**: Number counter for stats
- **Donation Form**: Custom form with frequency toggles
- **Volunteer Form**: Dedicated verification form
- **Bootstrap Icons**: Extensive use of BI icons
