# Technology template_a_v1

## Template Style: SEO Digital Agency
- **Framework**: Bootstrap 5.2
- **Typography**: Open Sans (Google Fonts)
- **Theme**: Dark header with modern corporate styling

---

## Sections

- **Pre-Header**: Contact info bar (Phone + Email + Address + Social Icons)
- **Header/Navbar**: Logo + Navigation (Home, Services, Projects, Pages dropdown, Infos, Contact)
- **Hero Section**: ID `#top` (Headline + Subtext + Dual CTAs: "Discover More" + "Check our FAQs")
- **Services**: ID `#services` (Section heading + 4-column grid with image icons)
- **Projects**: ID `#projects` (Owl Carousel slider showcasing portfolio items)
- **Infos/About**: ID `#infos` (Left image + Right content with skill bars + Text)
- **Contact**: ID `#contact` (Google Maps embed + Contact form with Name, Surname, Email, Subject, Message)
- **Footer**: Logo + Copyright + Credits
