# Construction template_a_v1

## Template Style: WEBUILD - Construction Company
- **Framework**: Bootstrap 5
- **Typography**: Open Sans + Roboto (Google Fonts)
- **Theme**: Dark radial gradient with yellow/gold primary accent

---

## Sections

- **Topbar**: 3-column info bar (Office Address, Email, Phone)
- **Navbar**: Dark sticky nav + Building icon logo + Navigation + "Get A Quote" CTA
- **Hero Carousel**: Fullscreen construction images + Headline + CTA
- **About Section**: Text content + Checkmarks (2-column) + Signature image + Offset photo
- **Services**: 6-card grid with images (Building Construction, House Renovation, Architecture Design, Interior Design, Fixing & Support, Painting)
- **Appointment/Callback**: Request callback form section
- **Portfolio/Projects**: Isotope filterable gallery (All, Construction, Renovation) + Lightbox
- **Team**: 4 team members with side social icons
- **Testimonials**: Owl Carousel + Large side image
- **Blog**: 3-column blog posts grid
- **Footer**: Dark radial gradient + 2-column (Company info + Links + Newsletter)
