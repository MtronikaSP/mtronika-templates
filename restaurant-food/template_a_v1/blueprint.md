# Restaurant Food template_a_v1

## Template Style: Restaurantly - Fine Dining Restaurant
- **Framework**: Bootstrap 5
- **Typography**: Open Sans + Playfair Display + Poppins (Google Fonts)
- **Theme**: Dark/Black background + Gold/Amber accents (#cda45e)

---

## Sections

- **Topbar**: Fixed topbar with phone + hours + language switcher
- **Header/Navbar**: Fixed header + Logo ("Restaurantly") + Navigation (Home, About, Menu, Specials, Events, Chefs, Gallery, Contact) + "Book a table" CTA
- **Hero Section**: ID `#hero` (Full-height video background + Welcome text + Headline + Dual CTAs: Our Menu + Book a Table + Video play button)
- **About Section**: ID `#about` (Left content with checklist + Right image)
- **Why Us Section**: ID `#why-us` (3 numbered boxes layout)
- **Menu Section**: ID `#menu` (Filter tabs: All, Starters, Salads, Specialty + 2-column menu items with images, price, ingredients)
- **Specials Section**: ID `#specials` (Left vertical tab menu + Right content with image/description)
- **Events Section**: ID `#events` (Swiper carousel with event cards: Birthday, Private, Custom Parties with pricing)
- **Book A Table**: ID `#book-a-table` (Reservation form: Name, Email, Phone, Date, Time, # People, Message)
- **Testimonials**: ID `#testimonials` (Swiper carousel with client quotes + images)
- **Gallery Section**: ID `#gallery` (8-image lightbox grid)
- **Chefs Section**: ID `#chefs` (3-column chef cards with social overlay)
- **Contact Section**: ID `#contact` (Address, Hours, Phone, Email + Google Map embed + Contact form)
- **Footer**: Social icons row + Copyright
