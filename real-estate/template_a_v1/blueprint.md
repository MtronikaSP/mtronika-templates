# Real Estate template_a_v1

## Template Style: Makaan - Real Estate Property
- **Framework**: Bootstrap 5
- **Typography**: Heebo + Inter (Google Fonts)
- **Theme**: Teal/Green primary (#00B98E) + Clean white background

---

## Sections

- **Spinner/Loader**: Animated loading spinner
- **Navbar**: Transparent nav + Icon logo ("Makaan") + Navigation dropdowns (Property, Pages) + "Add Property" CTA
- **Header/Hero**: Left content (Headline + Description + "Get Started" CTA) + Right Owl Carousel images
- **Search Bar**: Primary bg bar with Keyword, Property Type, Location selects + Search button
- **Property Types**: 8 category icon cards (Apartment, Villa, Home, Office, Building, Townhouse, Shop, Garage) with property counts
- **About Section**: Left bordered image + Right content with checklist + "Read More" CTA
- **Property Listing**: Tabbed listings (Featured, For Sell, For Rent) + 6 property cards per tab with images, price, location, Sqft/Bed/Bath details + "Browse More" CTA
- **Call To Action**: Bordered card with image + "Contact With Certified Agent" + dual CTAs (Make A Call, Get Appointment)
- **Team/Agents**: 4-column agent cards with social overlay icons
- **Testimonials**: Owl Carousel testimonial cards with client quotes
- **Footer**: Dark footer with Address column + Quick Links + Services + Newsletter signup + Copyright bar
